# LCA MySQL Exercises

This repository contains SQL scripts and exercises for the **EduTrack SA** database completed during the Backend Web Development course.

## Exercises

* **Week 1 - Exercise 01:** Database setup and schema creation for EduTrack SA.
* **Week 1 - Exercise 02:** Querying, sorting (`ORDER BY`, `LIMIT`), filtering (`WHERE`, `LIKE`), aggregate functions (`COUNT`, `AVG`, `MAX`), and grouping (`GROUP BY`, `HAVING`).
