# LCA MySQL Exercises

Repository for my MySQL exercises for Course 2 (Backend Web Development) at LCA. These projects focus on designing, normalizing, and querying relational databases using MySQL Workbench.

## Database Overview
The exercises are built around **EduTrack SA**, a Cape Town-based online learning platform.

---

## Exercises

### Exercise 01: Database Design and Table Creation
* **File:** `week1_mysql_ex01_YusufOsman.sql`
* **Branch:** `week1-mysql-ex-01`
* **Summary:** Designed and built a normalized relational database (`edutrack_sa`) in Third Normal Form (3NF). Created four connected tables (`facilitators`, `courses`, `trainees`, and `enrolments`) using primary/foreign keys and constraints, then populated them with realistic South African sample data.

### Exercise 02: Querying, Sorting, and Filtering Data
* **File:** `week1_mysql_ex02_YusufOsman.sql`
* **Branch:** `week1-mysql-ex-02`
* **Summary:** Wrote SQL queries to analyze and extract reports from `edutrack_sa`. Covers data sorting (`ORDER BY`, `LIMIT`), string filtering (`LIKE`), aggregate functions (`COUNT`, `AVG`, `MAX`), and grouped filtering (`GROUP BY`, `HAVING`).
