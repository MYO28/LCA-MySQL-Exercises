# Exercise 03: SQL Joins and Data Manipulation

This branch contains the SQL solutions for **Exercise 03** in the Backend Web Development course. The exercise focuses on querying relational data across multiple tables in the `edutrack_sa` database and performing Data Manipulation Language (DML) operations.

## Featured Technical Concepts

* **INNER JOIN:** Combining records across multiple tables (`trainees`, `enrolments`, `courses`, and `facilitators`).
* **LEFT & RIGHT JOIN:** Handling unmatched records and `NULL` values for unenrolled trainees and courses without active enrolments.
* **Data Manipulation (DML):** Updating existing record attributes with `UPDATE` and safely deleting records using `DELETE` with `ORDER BY` and `LIMIT`.
* **Multi-Clause Aggregations:** Combining `JOIN`, `WHERE`, `GROUP BY`, `HAVING`, and `ORDER BY` into a single analytical statement.
